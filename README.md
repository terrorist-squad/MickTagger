# MickTagger
Bash script for adding a tags  to calibre ebook database and converting PDFs in images (with OCR).

# Install
```
$ apt-get update && apt-get install -y xpdf calibre
$ chmod 755 blind-reader.sh
$ ./blind-reader.sh
```
